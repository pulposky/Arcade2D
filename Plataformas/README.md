# 🎮 Plataformas

Un juego de plataformas 2D desarrollado en **Python** y **Pygame**, donde el objetivo es sobrevivir, recolectar monedas y enfrentar enemigos en un mundo que aumenta progresivamente de tamaño y dificultad.

## ✨ Características

- 🏃 Movimiento lateral y salto.
- 🪙 Sistema de recolección de monedas.
- 👾 Enemigos que persiguen al jugador.
- ❤️ Sistema de vidas.
- ⚡ Power-ups temporales:
  - Aumento de velocidad.
  - Invencibilidad.
- 👑 Batallas contra jefes (Bosses).
- 📈 Incremento gradual de la dificultad.
- 🏆 Sistema de récord de puntuación.
- 🎵 Efectos de sonido y música.

## 🛠️ Tecnologías

- Python 3.13
- Pygame 2.6.1

## 🚀 Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/pulposky/Arcade2D.git
```

2. Accede a la carpeta del proyecto:

```bash
cd Arcade2D/Proyecto
```

3. Instala las dependencias:

```bash
pip install pygame
```

4. Ejecuta el juego:

```bash
python main.py
```

## 🎮 Controles

| Tecla | Acción |
|---------|---------|
| A / ← | Mover a la izquierda |
| D / → | Mover a la derecha |
| Espacio | Saltar |
| ESC | Pausar |

## 🎯 Objetivo

Consigue la mayor cantidad de puntos posible mientras esquivas enemigos, recoges monedas, aprovechas los power-ups y derrotas a los jefes que aparecerán durante la partida.
