# 🎮 Colección de Juegos 2D

Bienvenido a este repositorio, un espacio donde se almacenarán y desarrollarán diferentes **juegos 2D** creados con fines de aprendizaje, práctica y experimentación.

## 📚 ¿Qué encontrarás aquí?

Este repositorio incluirá proyectos de distintos géneros, tales como:

- 🏃 Juegos de plataformas
- 🃏 Juegos de cartas
- 🚀 Arcade
- 🧩 Puzles
- ⚔️ Acción y aventura
- 🎯 Otros proyectos experimentales

Cada juego contará con su propia carpeta, incluyendo el código fuente y los recursos necesarios para su funcionamiento.

## 🛠️ Tecnologías

Los proyectos se desarrollarán principalmente utilizando:

- Python
- Pygame
- Git y GitHub

## 🎯 Objetivo

Crear una colección de videojuegos 2D que permita poner en práctica conceptos de programación, diseño de mecánicas y desarrollo de software, mientras se construye un portafolio de proyectos.

---

⭐ Este repositorio seguirá creciendo con nuevos juegos y mejoras en los proyectos existentes.
